### Pré-requisitos
- Neo4j (4.4+)
- Biblioteca GDS instalada

### Instalação
1. Clone o repositório
2. Execute o schema: `cypher-shell -f data/schema.cypher`
3. Importe os dados: `cypher-shell -f data/dados_exemplo.cypher`
